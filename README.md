# demo-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### DEPLOY PROJECTS TO NAMECHEAP SERVER (CPANEL)
#### Check deploy script to copy files from your local computer
```
sh deploy/deploy.sh
```
